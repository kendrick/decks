Autoprefixer

Adds vendor prefixes based on caniuse.com compatibility info.

}

Can't Sass do this?

@mixin keyframes($name) {
  @-o-keyframes $name { @content };
  @-moz-keyframes $name { @content };
  @-webkit-keyframes $name { @content };
  @-keyframes $name { @content };
}

"Yes, but…"

Input 
.container {
  display: flex;
}

.container .column {
  display: flex;
  flex-direction: row;
}
Output 
.container {
  display: -webkit-box;
  display: -ms-flexbox;
  display: flex;
}

.container .column {
  display: -webkit-box;
  display: -ms-flexbox;
  display: flex;
  -webkit-box-orient: horizontal;
  -webkit-box-direction: normal;
      -ms-flex-direction: row;
          flex-direction: row;
}

Font Magician

Builds complicated @font-face rules automatically.

Input 
body {
  font-family: 'Work Sans';
}
Output 
@font-face {
  font-family: "Work Sans";
  font-style: normal;
  font-weight: 100;
  src: local("Work Sans Thin"),
       local(WorkSans-Thin),
       url(//fonts.gstatic.com/s/worksans/v2/[…].woff2)
         format("woff2"),
       url(//fonts.gstatic.com/s/worksans/v2/[…].woff)
         format("woff");
}

body {
  font-family: 'Work Sans';
}

cssnano

Extreeeeeeme minification of styles.

Input 
body {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
  transform: translate3d(0, 0, 0);
  background: url("./css/../img/cat.jpg");
}
Output 
body {
  margin: 10px 20px;
  transform: translateZ(0);
  background: url(img/cat.jpg);
}
